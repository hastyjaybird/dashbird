# Notes

Edit this file at `public/data/notes.md`. It is shown as plain text on the dashboard.

- **Personal / Work tiles:** `public/data/bookmarks-personal.json` and `public/data/bookmarks-work.json` — each is an array of up to 9 objects: `{ "word": "Label", "href": "https://…" }`.
- **Background:** replace `public/assets/dashboard-bg.jpg` to change the wallpaper.
- Set your browser **home / new tab** URL to this app (for example `http://127.0.0.1:8787/` or your `HOST_PORT`).
